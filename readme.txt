Программа создает из прилагательных и их переводных эквивалентов таблицы сочетаемости с существительными. Используемые языки: английский, немецкий, русский

Входные данные:
1. Массив NAMES. 
2. Для каждого прилагательного в корневой папке программы содержится файл с биграммами. Скачать файлы можно по ссылке http://storage.googleapis.com/books/ngrams/books/datasetsv2.html. Файлы должны содержать в названии первые две буквы каждого прилагательного из списка и соответствующие им языки.
3. В корневую папку устанавливаются Mystem (https://tech.yandex.ru/mystem/) и TreeTagger (http://www.cis.uni-muenchen.de/~schmid/tools/TreeTagger/) с файлами английского и немецкого языков
4. Файлы обработанных словарей, включаающие все пары языков (русский-английский, английский-русский, русский-немецкий, немецкий-русский, английский-немецкий, немецкий-русский). Файлы имеют расширение .txt. Вид внутри: 'слово;перевод,перевод,...\r\n'. В работе использованы обработанные файлы словарей от проекта XML Dictionary eXchange Format (http://dicto.org.ru/xdxf.html). Словари приложены к программе.

Запуск:
1. В коде программы в массив подставляются массивы, имеющие три элемента: прилагательное, краткое обозначение языка (eng - английский, ger - немецкий, rus - русский) и эквивалент в кириллице для прилагательных русского языка. Для прилагательных на других языках третий элемент должен содержать строку 'none'.
2. После запуска программы пользователь получает информационные сообщения с временем старта и конца обработки определенного слова.

Этапы работы программы:
1. Обработка файла с биграммами для каждого отдельного прилагательного.
	а) Обработка исходного файла биграмм от Google: выделение словосочетаний, состоящих из прилагательного с существительным по разметке, присутствующей в исходном файле
	б) Использование морфологических анализаторов
	в) Обработка файлов в соответствии с результатами проверки морфологическими анализаторами
2. Сведение полученных на первом этапе файлов с биграммами в таблицы, показывающие сочетаемость прилагательных и их переводных эквивалентов

Исходные данные:
В корневой папке появляется папка 'final', содержащая папки с файлами, отображающими результаты работы программы на каждом этапе. Итоговые таблицы находятся в папке '4_translated'.
Помимо этого, в папке 'final' находится папка 'not_translated', содержащая списки существительных для каждого прилагательного, для которых не найдены переводы в словарях.
